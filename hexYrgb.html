Hex y RGB: Colores Web

Hexadecimal

En CSS, Hex se refiere a un formato de notación hexadecimal utilizado para definir colores. El formato hexadecimal utiliza el sistema numérico base 16, que incluye los números del 0 al 9 y las letras de la A a la F. Cada color en hexadecimal se representa con un # seguido de seis caracteres, donde cada par de caracteres define la intensidad del color rojo, verde y azul (RGB), respectivamente.

Estructura de un Color Hexadecimal

#RRGGBB

  • RR representa el componente rojo.
  • GG representa el componente verde
  • BB representa el componente azul.

RGB

En CSS, RGB es un modelo de color que se utiliza para definir colores mediante la combinación de los tres colores primarios de luz: rojo (Red), verde (Green) y azul (Blue). Cada componente puede tener un valor que va de 0 a 255. Estos valores indican la intensidad de cada color.

Sintaxis de RGB

La sintaxis para definir un color usando RGB en CSS es:

Color: rgb(red, green, blue);

Ejemplos

  • rgb(255, 0, 0): Rojo (máxima intensidad de rojo, sin verde ni azul).
  • rgb(0, 255, 0): Verde (máxima intensidad de verde, sin rojo ni azul).
  • rgb(0, 0, 255): Azul (máxima intensidad de azul, sin rojo ni verde).
  • rgb(255, 255, 255): Blanco(máxima intensidad de todos los colores).
  • rgb(0, 0, 0): Negro (sin intensidad de ninguno de los colores).
