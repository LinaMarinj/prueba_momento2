¿CSS?

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento escrito en HTML o XML. Permite controlar el diseño y la apariencia visual de una página web, incluyendo aspectos como colores, fuentes, márgenes y espaciado.

Aplicación de CSS

  1. En línea: Directamente en los elementos HTML usando el atributo style.
  2. Interno: Dentro de una etiqueta <style> en el documento HTML
  3. Externo: En archivosH.css separados vinculados al HTML HTML
