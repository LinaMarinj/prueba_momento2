Especificidad de Estilos

La especificidad en CSS es un sistema que determina qué reglas CSS se aplican a un elemento cuando hay múltiples reglas que podrían aplicarse. La especificidad se basa en el número de cada tipo de selector en la regla y sigue una jerarquía de importancia.

Conceptos Clave de Especificidad

La especificidad se calcula en base a una puntuación que se asigna a cada regla CSS. Esta puntuación se compone de cuatro valores, que se podrían conceptualizar como: (a, b, c, d).

  • a: Estilos en línea (style="..."). Tienen la especificidad más alta.
  • b: Selectores de ID (#id)
  • c: Selectores de clase, atributos y pseudo-clases (.class, [attribute], :hover, etc.).
  • d: Selectores de tipo (elementos HTML) y pseudo-elementos (div, p, ::before, etc.)

Cómo se Calcula la Especificidad

Se suma el número de cada tipo de selector en la regla. La especificidad se expresa como un conjunto de cuatro valores (a, b, c, d), donde cada valor tiene un peso diferente, y los valores más altos tienen mayor prioridad.
